

В рамках ДЗ сделано 5 классов, все они выполняют конкретную функцию:
1) BaseTest - управляет конфигурацией тестов, является базовым шаблоном.
Указывает, как записывать логи и чистит тестовые данные после всех тестов.

2) UserAccount - готовит тестовые данные от имени админа и самих юзеров.
Результат его работы - это юзер с открытым аккаунтом.
Также он дает доступ через геттеры всем, кто захочит с ними работать.
+ дает возможность чистить все тестовые данные(изначально делал в ручную в сваггере).

3) AccountTransferTest, DepositAccountTest, UserChangesUsernameTest = тестовые классы.
Внутри каждого теста создаешь независимых друг от друга юзеров.
По итогу AfterAll зачистит все следы, а по логам выполнения можно посмотреть на все, что происходило под капотом.
